## 📋 What is ai-studio-bridge?

ai-studio-bridge is a tool that helps you use Google AI Studio in an easier way. It works behind the scenes to make things automatic. You do not need to open Google AI Studio yourself. The tool can help with:

- Sending commands to Google AI Studio.
- Extracting files or data automatically.
- Managing AI models through the app.

It runs on your computer and uses a software called Playwright to open and control the web pages for you. This turns the Google AI Studio web interface into something you can control with easy commands, without needing to visit the website each time.

This tool is safe for use with multiple processes at the same time (thread-safe). It allows people who want to build their own apps or projects using Google AI Studio to have an easy way to connect to it.
